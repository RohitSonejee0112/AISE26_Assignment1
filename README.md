## Project Overview

This project is a Kotlin-based console application that simulates the FIFA World Cup 2026 tournament with an integrated betting system.

Users can:

- View group standings
- Browse match fixtures
- Place bets on matches
- Track wins, losses, and profits
- Progress through knockout rounds
- Simulate the tournament until the final

---

## Main Features

- Full tournament flow from Group Stage to Final
- Realistic match results and betting odds
- Betting system with payout calculations
- Team statistics and standings
- Knockout bracket progression
- Console-based interactive menus
- JSON data loading using `kotlinx.serialization`
